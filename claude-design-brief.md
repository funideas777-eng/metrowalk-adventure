## Prompt 1:活動海報 (A4 直式)

請幫我設計一張 **A4 直式 (210×297mm) 活動海報**,主題是企業團隊闖關冒險遊戲。

**活動名稱**:大江購物冒險 Metrowalk Adventure
**副標**:TSMC BSID 600 人科技闖關大冒險
**時間**:2026 年 5 月 1 日 10:00–14:00
**地點**:大江購物中心(桃園市中壢區中園路二段 501 號)
**規模**:60 隊 × 10 人 = 600 人

**視覺風格**(務必沿用,與我們現有 APP UI 一致):
- 主色:Magenta/Pink `#E91E63`
- 副色:Cyan `#00BCD4`
- 強調:Gold `#FFD740`
- 背景:Deep Navy `#0a0a1a`,漸層 135deg
- 質感:Glassmorphism 半透明卡片、cyan/pink 發光邊緣
- 圓角 14–16px、科技感、高能量、年輕感(非嚴肅企業風)
- 字體:粗體 sans-serif(標題 700–900 重量)

**畫面元素**:
1. 上方:超大標題「大江購物冒險」+ 副標
2. 中段:6 樓層 icon(GBF / 1F–5F)圍繞中央 hero 區
3. 下段:12 款關卡 emoji 網格(👾🐍🧺🔨🃏🧱🧠🏹💎🎨🎵🧩)+ 5 款料理挑戰(🧋🍜🥟🥭🍍)
4. 底部:日期、地點、QR code 預留區塊
5. Call-to-action:「準備好展開一場科技冒險了嗎?」

請確保列印清晰(CMYK 友善、遠觀可辨識)。

## Prompt 2:規則說明單張 (A4 直式,一頁)

請幫我設計 **A4 直式一頁規則說明單張**,搭配上面的海報同一套視覺風格。

**必須包含的四大區塊**(一頁內清楚排版):

### 1. 活動流程 / Flow
- 10:00 報到 → 分隊(60 隊)→ APP 登入
- 10:15 開始闖關(App 地圖導航)
- 各樓層完成小遊戲、料理、團體任務累積分數
- 13:30 最後衝刺 / 緊急任務
- 14:00 結算頒獎

### 2. 計分規則 / Rules
- **小遊戲**:每關 100–200 分門檻,單關最高 5,000–8,000 分;3 條命、45–120 秒
- **隊伍名次加分**:第 1 名 +500 / 第 2 名 +400 / 第 3 名 +300 / 第 4 名 +200 / 第 5 名 +100
- **料理任務 (5 道)**:完成食譜小遊戲 + 上傳試吃照片 = +250 分/次
- **團體照任務 (3 種)**:愛心隊形 / 跳躍照 / 人體金字塔 = +250 分/次
- **緊急任務**:Admin 隨機觸發快閃挑戰 +300–500 分
- **GPS 解鎖範圍**:100m 內自動解鎖關卡

### 3. 關卡圖示說明 / Game Icons
以 icon + 名稱網格呈現 14 款關卡:
- 👾 吃豆人 / 🐍 貪食蛇 / 🧺 接接樂 / 🔨 打地鼠 / 🃏 翻牌配對
- 🧱 打磚塊 / 🧠 知識問答 / 🏹 射箭高手 / 💎 晶圓大師
- 🎨 顏色陷阱 / 🎵 節奏大師 / 🧩 拼圖挑戰
- 🍳 料理冒險 / 📸 團體任務

### 4. 交通路線 / Transportation
- 地址:桃園市中壢區中園路二段 501 號
- 簡易地圖示意(從中壢火車站 / 中壢交流道方向)
- 停車資訊:大江 B1–B3 停車場
- 建議:搭乘火車到中壢站轉公車或步行約 15 分鐘
- QR code 預留(Google Maps 導航連結)

**風格要求**:
- 與海報同色系(pink/cyan/gold/navy)
- 使用 icon + 分區 card 排版,避免大段文字
- 正體中文,語氣活潑(搭配 emoji)
- 資訊密度高但要留白,一眼能找到想看的區塊
